What This Enables

Once the credentials are saved, you no longer need to pass Adzuna keys on every CLI run.

  1. python3 -m offerquest refresh-jobs can rebuild the configured job-source files using the saved credentials.
  2. python3 -m offerquest fetch-adzuna can also fall back to the saved file when flags are omitted.
  3. The credentials file stays outside the workspace, so it is not committed into this repo or recorded in workbench runs.
