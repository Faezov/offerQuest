Adzuna

App id and app key

  1. Create or sign in to an Adzuna developer account.
  2. Register your API use so Adzuna issues an app_id and app_key.
  3. Copy those values from your Adzuna account details or API dashboard and save them here.

Official docs: Overview, Register, Account details

Greenhouse

Board token, not a private API key

OfferQuest reads Greenhouse jobs through the public Job Board API, so for this source type you usually need the job board token, not a secret key.

  1. In Greenhouse Recruiting, open Job Boards & Posts and edit the target board settings.
  2. Find the board URL. The last segment of https://boards.greenhouse.io/example-company is the token.
  3. Paste only that last segment into the board_token field for your Greenhouse source.

Official docs: Job Board API, Find your job board URL, Board token guide
