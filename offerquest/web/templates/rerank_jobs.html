What This Phase Does

The rerank keeps the existing ranking format but adds more ATS-oriented context for the top window.

  1. Starts from the current ranking context and chosen jobs file.
  2. Builds ATS reports for the top window using the selected CV and optional cover letter.
  3. Combines first-pass fit with ATS score and required-keyword coverage.
  4. Saves a reranked JSON artifact with movement, score deltas, and short rerank reasons.
