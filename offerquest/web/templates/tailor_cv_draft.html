What This Phase Does

This creates a concrete draft, not just a plan.

  1. Rewrites the top of the resume using the phase-1 tailoring logic.
  2. Reorders visible skills and adds only safe keywords already supported by the candidate documents.
  3. Shows ATS before and after, then saves the tailored text plus a comparison JSON artifact.
  4. Optionally exports a plain ATS-safe .docx version from the same draft.
